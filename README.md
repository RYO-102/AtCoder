# AtCoder Solutions & Algorithm Templates

AtCoderの解答コードおよび、競技プログラミングにおける汎用アルゴリズムのテンプレートを管理するリポジトリです。

## 概要

本リポジトリは、単なるソースコードの蓄積に留まらず、外部管理ツールを併用した「思考プロセスの言語化」と「アルゴリズム別の検索性」の両立を目的としています。実務におけるドキュメンテーション能力と課題解決の再現性を意識したワークフローを構築しています。

## 学習管理システム

解答済みの問題は、専用のスプレッドシートにて一括管理しています。
[AtCoder_Problem_Management_Log (RYO102)](https://docs.google.com/spreadsheets/d/1i_ehQYstFpRDq0EW1m6yQqwY_5taHa1L8dnhylnPYMc/edit?usp=sharing)

### 管理対象の基準
検索性の維持と学習精度の向上を目的に、以下の基準で記録を行っています。
- **ABC (AtCoder Beginner Contest)**: 原則として考察を要する **C問題以上** を対象。
- **ARC (AtCoder Regular Contest)**: 全問題を対象。
- **典型・書籍演習**: 「競プロ典型90問」や「競技プログラミングの鉄則」などの体系的演習は、全問題を対象。

### 管理項目と運用の工夫
- **思考プロセスの言語化**: 解法に至るまでの考察順序を詳細に記録。セル内の配置を最適化し、長文の考察も即座に俯瞰できる構成にしています。
- **多角的な検索**: アルゴリズム、キーワード、類似アプローチの迅速な検索を可能にしています。
- **厳密なステータス管理**: 「判定（AC/解説AC/復習中）」や「実施日」を選択式（入力規則）で管理。未習得アルゴリズムの早期発見と重点的な復習に活用しています。

## ディレクトリ構成

* **ABC/**: AtCoder Beginner Contest の解答（コンテスト別管理）。
* **ARC/**: AtCoder Regular Contest の解答。
* **ATC/**: AtCoder Typical Contest 等、典型アルゴリズムの演習。
* **競プロ典型90問/**: 典型的な問題パターン 90 問の演習。
* **競技プログラミングの鉄則/**: 書籍『競技プログラミングの鉄則』に基づく演習。
* **アルゴリズムテンプレート/**: 汎用的なデータ構造やアルゴリズムのスニペット群。
